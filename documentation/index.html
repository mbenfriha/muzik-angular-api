API VERSION : 1.0

Albums: tout sur les albums voir/caché
[GET] /albums Liste tous les albums
[GET] /albums/{id} Liste un album avec ses musiques par l'ID
[GET] /albums/artists{id} Liste tous les albums d'un artistes par l'ID
[GET] /albums/rand Liste 10 albums au hasard
Artistes: accéder aux artistes voir/caché
[GET] /artists Liste tous les artistes
[GET] /artists/{id} Retourne un artiste
Genres: lister les genres voir/caché
[GET] /genres Liste tous les genres
[GET] /genres/{id} Liste les albums du genre par l'ID
